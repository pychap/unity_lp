Go to Mars!

The Vuforia Mars sample was inspired by NASA’s Mars Journey missions and premiered live at the 2017 Unity Vision Summit.

The Mars Sample demonstrates best practices for games designed to be played in the real-world.

This sample includes rich assets and animations that you can use to create your own experiences.
